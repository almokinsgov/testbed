FNED live export bundle

Open:
- index.html (recommended)
- v23_v32_split_js_v2.html

Recommended:
- Serve with a local static server (e.g. VS Code Live Server) to avoid browser file:// restrictions.

- src/data/endpoints.js
- src/data/fetch_helpers.js
- files/public_closures_FNDC_data.js
- files/fned_custom_messages_example.json

iPhone install steps for notifications:

1. Open the dashboard in Safari
2. Share then Add to Home Screen
3. Open FNED from the Home Screen icon
4. Tap Receive Notifications

iPhone Chrome note:
- iPhone Chrome cannot install to Home Screen directly. Use Safari for the install steps.
